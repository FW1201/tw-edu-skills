# 素養導向教案標準格式規範

## 教案格式原則
1. 以表格為主要呈現結構
2. A4 直印，上下邊距 2cm，左右邊距 2.5cm
3. 字型：標楷體（標題 16pt 粗體；表頭 12pt 粗體；內文 12pt）
4. 主色：教育部藍 #1A5276；表頭底色 #2471A3；奇數列 #EBF5FB
5. 封面頁需含：學校名稱、科目、課文/主題、年級班別、節數、設計者、日期

## 表格六：教學活動設計（核心表格）

### 每節課格式
```
節次：第 X 節（45 分鐘）
┌────────────┬────────────────────────────┬──────┬──────────┐
│ 教學流程   │ 教學活動詳細步驟             │ 時間 │ 評量方式 │
├────────────┼────────────────────────────┼──────┼──────────┤
│ 準備活動   │ 引起動機詳細步驟             │ 5分  │          │
│（5-10分）  │ 含教師問題與學生預期回應      │      │          │
├────────────┼────────────────────────────┼──────┼──────────┤
│ 發展活動   │ 【一、主活動名稱】（X分鐘）  │ 35分 │ 觀察/     │
│（30-35分） │ 具體步驟1                   │      │ 學習單/   │
│            │ 具體步驟2                   │      │ 口頭問答 │
│            │ 具體問題句（直接寫出）       │      │          │
│            │ 【二、次活動名稱】（X分鐘）  │      │          │
│            │ ……                          │      │          │
├────────────┼────────────────────────────┼──────┼──────────┤
│ 綜合活動   │ 小結要點                    │ 5分  │          │
│（5分）     │ 預告下節/課後任務            │      │          │
└────────────┴────────────────────────────┴──────┴──────────┘
```

### 品質標準（必達到）
- 準備活動有具體的引起動機設計（非「教師介紹課文」）
- 發展活動至少 2 個子活動，各有子標題
- 每節課有至少 3 個具體師生問答問題（直接寫出問題句）
- 學生任務說明清楚：個人/小組/全班、時間多長
- 時間分配細化：每個子活動分鐘數加總等於該流程總時間
- 列出需要準備的教學媒材（學習單、投影片、圖卡等）
