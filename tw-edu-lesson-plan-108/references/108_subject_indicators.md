## 布魯姆認知分類（教學目標撰寫用）

| 層次 | 動詞舉例 |
|------|---------|
| 記憶（Remember） | 認識、列舉、說出、背誦、辨識 |
| 理解（Understand） | 解釋、描述、摘要、舉例、分類 |
| 應用（Apply） | 使用、執行、計算、解決、示範 |
| 分析（Analyze） | 區分、比較、分析、推論、判斷 |
| 評鑑（Evaluate） | 評估、批判、辯護、選擇、支持 |
| 創造（Create） | 設計、建構、規劃、產出、發明 |
