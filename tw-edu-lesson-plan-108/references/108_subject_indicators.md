### 第一學習階段（國小 1-2 年級）學習表現

#### 聆聽（1）
| 代碼 | 說明 |
|------|------|
| 1-Ⅰ-1 | 能養成良好的聆聽習慣，聆聽時集中注意力，不隨便打斷他人說話。 |
| 1-Ⅰ-2 | 能聆聽並理解他人說話的主要內容，對不理解的地方能提出詢問。 |

#### 口語表達（2）
| 代碼 | 說明 |
|------|------|
| 2-Ⅰ-1 | 能以清楚的發音說出完整的句子，表達自己的想法與感受。 |
| 2-Ⅰ-2 | 能在班級活動中，用簡單的語言分享自己的生活經驗。 |

#### 標音／識字與寫字（3）
| 代碼 | 說明 |
|------|------|
| 3-Ⅰ-1 | 能正確認識注音符號，準確拼讀，並用注音輔助識字。 |
| 3-Ⅰ-2 | 能依照正確筆順書寫基本筆畫與常用部件，並書寫常用字。 |
| 3-Ⅰ-3 | 能認識並書寫生活中常用字詞，累積識字量（約 500 字）。 |

#### 閱讀（5）
| 代碼 | 說明 |
|------|------|
| 5-Ⅰ-1 | 能透過圖文並茂的文本，理解文章的主要內容與情節。 |
| 5-Ⅰ-2 | 能認識文本的基本形式（故事、兒歌、短文），初步理解不同文類特色。 |
| 5-Ⅰ-3 | 能朗讀短文，注意正確的斷句與語調，感受語文的韻律美感。 |

#### 寫作（6）
| 代碼 | 說明 |
|------|------|
| 6-Ⅰ-1 | 能書寫完整的句子，表達個人的觀察與感受。 |
| 6-Ⅰ-2 | 能依提示或圖片，寫出簡短的短文，內容有頭有尾。 |

### 第一學習階段學習內容

| 代碼 | 說明 |
|------|------|
| Aa-Ⅰ-1 | 注音符號的聲母、韻母、聲調及結合韻的正確拼讀。 |
| Ab-Ⅰ-1 | 生活常用字詞的音、形、義（含同音字的基本辨識）。 |
| Ac-Ⅰ-1 | 簡單記敘性文本的人、事、時、地、物等基本要素。 |
| Ba-Ⅰ-1 | 常用標點符號（句號、逗號、問號、驚嘆號）的功能與用法。 |
| Ca-Ⅰ-1 | 常見修辭：比喻、擬人的初步認識。 |
