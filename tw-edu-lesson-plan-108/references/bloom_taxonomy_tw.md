# 布魯姆認知分類法（Bloom's Taxonomy）中文版

## 六個認知層次（由低到高）

| 層次 | 英文 | 代碼 | 核心動詞（教學目標用） |
|------|------|------|----------------------|
| 1 記憶 | Remember | C1 | 說出、列舉、認識、辨識、背誦、複述 |
| 2 理解 | Understand | C2 | 解釋、描述、摘要、分類、舉例、改寫 |
| 3 應用 | Apply | C3 | 使用、執行、計算、解決、示範、應用 |
| 4 分析 | Analyze | C4 | 比較、分析、推論、區分、組織、歸納 |
| 5 評鑑 | Evaluate | C5 | 評估、批判、辯護、選擇、判斷、支持 |
| 6 創造 | Create | C6 | 設計、建構、規劃、組合、創作、發明 |

---

## 素養導向教案中的應用策略

### 各層次對應教學活動
| 層次 | 適合的課堂活動 |
|------|--------------|
| C1 記憶 | 背誦練習、字詞填空、字音字形測驗 |
| C2 理解 | 課文大意摘要、段落解釋、概念圖 |
| C3 應用 | 造句練習、仿作練習、情境應用題 |
| C4 分析 | 文章結構分析、主題討論、比較閱讀 |
| C5 評鑑 | 閱讀心得、評論寫作、辯論活動 |
| C6 創造 | 創意寫作、故事改編、跨域設計 |

### 108 課綱素養導向建議
- **基礎能力**（C1-C2）：字詞教學、文意理解、概念建立
- **中層思維**（C3-C4）：應用練習、分析討論、跨文本比較
- **高階思維**（C5-C6）：批判評鑑、創意表達、議題探討

素養導向教案應兼顧各層次，但以 C3-C5 為主要設計重心。

## 教學目標撰寫格式（ABCD 法）
```
A（受眾 Audience）： 學生
B（行為 Behavior）： 能 [動詞] [具體行為]
C（條件 Condition）：在 [情境/資源] 下
D（標準 Degree）：  達到 [標準/程度]
```

**範例：**
- 「學生能正確說出《背影》中的 5 個生難字詞的注音與字義。」（C1）
- 「學生能以 3-5 句話摘要課文的主要情節。」（C2）
- 「學生能仿照課文的動作描寫，寫出一段 80 字以上的親情片段。」（C3）
- 「學生能比較《背影》與《鄉愁》中，作者表達情感的方式有何異同。」（C4）
