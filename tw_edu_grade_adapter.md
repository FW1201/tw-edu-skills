## 一、年級段落對照表

```
國小低年級  → 1-2 年級（第一學習階段）
國小中年級  → 3-4 年級（第二學習階段）
國小高年級  → 5-6 年級（第三學習階段）
國中        → 7-9 年級（第四學習階段）
高中        → 10-12 年級（第五學習階段）
```

## 二、觸發偵測關鍵詞

Claude 應從使用者輸入自動辨識學習階段：

| 輸入詞語 | 對應學習階段 | 代碼後綴 |
|----------|------------|---------|
| 國小一/二年級、低年級、一年、二年 | 第一學習階段 | -E- ／ 階段 I |
| 國小三/四年級、中年級、三年、四年 | 第二學習階段 | -E- ／ 階段 II |
| 國小五/六年級、高年級、五年、六年 | 第三學習階段 | -E- ／ 階段 III |
| 國中、七年、八年、九年、七八九年級 | 第四學習階段 | -J- ／ 階段 IV |
| 高中、高一、高二、高三、十年、十一年、十二年 | 第五學習階段 | -U- ／ 階段 V |

## 三、各學習階段內容適應規則

### 🟢 國小低年級（第一學習階段）
- **語言**：句子短、詞彙簡單、多用注音輔助
- **提問**：字面理解為主（布魯姆 C1-C2）
- **活動**：圖文並茂、遊戲化、具體操作
- **字數**：學習單留白大（每題作答空間 ≥ 3 行）
- **評量**：口語觀察 + 圖畫 + 簡單填空
- **課綱術語**：使用「能…」的行為目標，避免艱深術語

### 🔵 國小中高年級（第二、三學習階段）
- **語言**：中等複雜度，偶爾出現四字詞語
- **提問**：字面 + 推論（布魯姆 C1-C4）
- **活動**：小組合作、概念圖、簡短閱讀理解
- **字數**：每題作答空間 2-3 行
- **評量**：學習單 + 口頭 + 小測驗
- **課綱代碼**：使用 -E- 學習表現

### 🟡 國中（第四學習階段）
- **語言**：標準複雜度，含文言詞彙（國語文）
- **提問**：三層次提問（字面/推論/評鑑/創造）
- **活動**：分析討論、議題探究、仿作練習
- **字數**：仿作 100-150 字，問答 80-120 字
- **評量**：多元評量（紙筆 + 口語 + 作品）
- **課綱代碼**：使用 -J- 學習表現

### 🔴 高中（第五學習階段）
- **語言**：高度複雜，含文言文、跨語言詞彙
- **提問**：高階思維（C4-C6：分析/評鑑/創造）
- **活動**：思辨討論、跨文本比較、批判性寫作
- **字數**：議論文 400-600 字，問答 150-200 字
- **評量**：學習歷程導向、真實性評量
- **課綱代碼**：使用 -U- 學習表現

## 四、自動偵測 + 詢問流程

每個 skill 啟動時，Claude 應執行以下流程：

```
Step A：從使用者輸入中嘗試自動偵測年級
  → 若偵測成功：「我偵測到您的學生是[年級]，以下資訊請確認或修正……」
  → 若無法偵測：直接詢問（見下方提問範例）

Step B：確認後，自動調整以下所有輸出：
  - 教學語言難度
  - 布魯姆認知層次分布
  - 活動設計複雜度
  - 課綱代碼學段後綴
  - 評量方式建議
  - 文件排版（字號、留白）
```

**偵測成功範例：**
```
使用者：「幫我做一份《論語》教案，高中一年級」
→ Claude 自動設定：高中 / 第五學習階段 / -U- / C4-C6 層次
→ 不再詢問年級，直接進入其他資訊收集
```

**偵測失敗範例：**
```
使用者：「幫我出一份數學考卷，二次方程式」
→ Claude：「請問這份試卷要給幾年級的學生？（國小/國中/高中）」
```

## 五、難度標記系統（試卷、學習單共用）

| 符號 | 難度 | 布魯姆層次 | 適用學段 |
|------|------|-----------|---------|
| ★☆☆ | 基礎 | C1-C2（記憶/理解） | 各學段均有 |
| ★★☆ | 中等 | C3-C4（應用/分析） | 國小高 ～ 高中 |
| ★★★ | 進階 | C5-C6（評鑑/創造） | 國中 ～ 高中為主 |

試卷難度分布建議：
- 國小：★☆☆ 60% / ★★☆ 35% / ★★★ 5%
- 國中：★☆☆ 40% / ★★☆ 45% / ★★★ 15%
- 高中：★☆☆ 20% / ★★☆ 45% / ★★★ 35%
