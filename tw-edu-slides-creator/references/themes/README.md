# 教學簡報主題庫

本目錄包含預設主題，可在 Step 0 主題選擇時直接套用。

## 使用方式

選定主題後，Claude 讀取對應 `.md` 檔案，直接套用其 CSS 變數與字型設定，**跳過 Step 2 的顏色風格問答**（仍詢問頁數、文字密度、動態/靜態）。

## 可用主題

| 主題 | 適用場合 |
|------|---------|
| `edu-warm` | 一般教學、國語文、社會、藝術、SEL |
| `tech-dark` | AI 課程、程式設計、資訊科技、期末發表 |
| `academic-clean` | 研究報告、碩論簡報、正式發表、高中學術 |

> 若無主題符合需求，選「自訂」→ 進入 Step 2 自行描述風格。
