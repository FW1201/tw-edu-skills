# Sample Approval Template

Use this checklist when showing the sample slide.

```markdown
樣張檢查

- Slide: {slide number and title}
- 年級：{grade band}
- Backend: {image backend/tool}
- Path: `{origin_image/slide_XX.png}`

請確認：
1. 這張投影片對目標年級會不會太難或太滿？
2. 中文字是否清楚、正確、夠大？
3. 視覺是否真的幫助理解，而不是裝飾？
4. 示例、解析、活動的順序是否符合教學現場？
5. 是否需要調整色彩、圖像語言、文字量或活動形式？

確認後我會用同一套風格生成剩餘頁面。
```
