# 快速上手指南

> 5 分鐘內開始使用臺灣 K-12 教育 Skills 套組

---

## 安裝（一次搞定）

```bash
git clone https://github.com/YOUR_USERNAME/tw-edu-skills.git
cd tw-edu-skills
bash install.sh
pip install -r requirements.txt
```

安裝完成後，技能會出現在 `~/.claude/skills/` 目錄中。

---

## 使用方式

打開 Claude Code，直接用中文說你要什麼：

### 🏫 教案生成（最常用）
```
幫我做一份《背影》的教案，國中八年級，翰林版，3節課
```
→ 自動生成 **9 大表格** 的完整教案 `.docx`，含學習目標、活動設計、評量規劃

---

### 📊 課程地圖
```
幫我規劃國中八年級國語文上學期的課程地圖，單元包括：背影、鄉愁、岳陽樓記、木蘭詩
```
→ 生成 **4 個工作表** 的 `.xlsx`，含週次規劃、課綱對應、跨域連結

---

### 📝 試卷出題
```
幫我出一份國中九年級國語文《背影》段考卷，100分，含選擇20題、填充10格、問答3題
```
→ 產出 **試卷 + 答案卷** 兩份 `.docx`

---

### 📋 評量規準
```
設計一份高中一年級議論文寫作的評量規準，分析式，5個向度，4等第
```
→ 完整分析式 Rubric，含學生自評與同儕評量欄位

---

### 📚 學習單
```
幫我設計一份《岳陽樓記》的課堂學習單，包含閱讀前啟動、三層次提問、概念圖和仿作鷹架
```
→ 美觀排版的素養導向學習單 `.docx`

---

### 📣 親師溝通
```
幫我寫一封戶外教學通知，七年一班，下週四去故宮，需要家長準備便當和帽子
```
→ 正式格式的家長通知單，含回條虛線

---

### 🤖 AI素養教學
```
幫我設計一個給國中生的提示詞工程入門教學簡報，45分鐘
```
→ 美觀的教學 `.pptx`，含活動設計與學生練習頁

---

### 📖 數位敘事
```
幫我做一份「我的家鄉故事」數位敘事設計教學簡報，適合高一
```
→ 含三幕劇結構、多媒體腳本模板的 `.pptx`

---

### 🎥 直式短影音學習動畫
```
幫我把國中自然《光合作用》做成 9:16 直式短影音，先給我分鏡、5 種動態流程和 5 種影片風格，再幫我設定 ElevenLabs 旁白
```
→ 先輸出表格 + 卡片分鏡，確認後再進入 Remotion 與語音設定

## 輸出位置

所有生成的文件會自動儲存並顯示下載連結。  
你也可以在命令中指定路徑：
```
幫我把教案儲存到桌面
```

## 常見問題

**Q: 可以一次做全班 30 位學生的評語嗎？**  
A: 可以！告訴 Claude 每位學生的表現描述，它會批量生成評語。

**Q: 生成的文件可以修改嗎？**  
A: 當然可以。所有輸出都是標準 .docx/.xlsx/.pptx，可用 Word / Excel / PowerPoint 自由編輯。

**Q: 支援英語教學嗎？**  
A: 目前以中文科目為主，英語科目使用通用教案格式，持續改善中。

**Q: 課綱代碼對不對我怎麼知道？**  
A: Claude 會顯示選用的課綱代碼及理由，你可以在文件中直接修改。
