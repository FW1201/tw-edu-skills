# 臺灣 108 課綱素養情境命題設計指南

## 素養命題 vs 傳統命題

| 面向 | 傳統命題 | 素養命題 |
|------|---------|---------|
| 情境 | 無（直接問知識） | 取材真實生活情境 |
| 測驗 | 記憶與背誦 | 知識整合與應用 |
| 認知 | 偏 C1-C2 | 偏 C3-C6 |
| 文本 | 單一連續文本 | 多元（含非連續） |
| 場景 | 學校學科情境 | 生活、社會、全球情境 |

## 情境來源原則（OECD PISA 框架）

### 個人情境
- 健康與體適能、個人休閒興趣
- 個人財務（零用錢、消費決策）
- 家庭生活與人際關係

### 社會情境（推薦用於國中高中）
- 社區生活、公民參與
- 健康/環境/科技的公共議題
- 媒體報導、統計數據判讀

### 全球情境（高中為主）
- 全球化議題（氣候、人口、移民）
- 科技倫理（AI、生技、隱私）
- 跨文化理解

### 本土情境（臺灣特色）
- 臺灣歷史文化
- 本土生態環境（玉山、日月潭、濕地）
- 臺灣科技產業（半導體、ICT）
- 原住民族文化

## 非連續文本題型設計指引

非連續文本 = 圖表、海報、時刻表、菜單、廣告、說明書、地圖等視覺材料

### 製作方式（文字模擬）
```
[圖表類型：折線圖]
標題：2020-2024 年臺灣各縣市空氣品質指標(AQI)趨勢
X 軸：年份（2020-2024）
Y 軸：AQI 數值（0-200）
資料系列：台北市（藍線）、高雄市（紅線）、花蓮縣（綠線）
關鍵數值：
・台北市 2020: 45 → 2024: 38（下降趨勢）
・高雄市 2020: 68 → 2024: 55（改善明顯）
・花蓮縣 2020: 28 → 2024: 25（持續最低）
```

### 對應題目示例
```
X. 依據上方折線圖：
（1）哪個縣市的空氣品質在 2020-2024 年改善幅度最大？（2分）
（2）若你是高雄市市長，根據圖表趨勢，你會提出什麼政策建議？（3分）
```

## 各年段難度分布參考值

### 國小（第一至三學習階段）
- ★☆☆（基礎）：60-70%
- ★★☆（中等）：25-35%
- ★★★（進階）：5-10%

### 國中（第四學習階段）
- ★☆☆（基礎）：35-45%
- ★★☆（中等）：40-50%
- ★★★（進階）：10-20%

### 高中（第五學習階段）
- ★☆☆（基礎）：15-25%
- ★★☆（中等）：40-50%
- ★★★（進階）：30-40%
