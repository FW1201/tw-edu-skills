## Step 1：資訊收集（最多問 3 個）

```
Q1: 考試範圍？（課文名稱、單元或主題）
Q2: 哪個年級、哪個科目？
Q3: 題型組合偏好？（預設見下表）
```

**預設題型組合**：

| 學習階段 | 題型分配 |
|---------|---------|
| 國小 | 選擇 40 + 填充 20 + 問答 40 = 100 分 |
| 國中 | 選擇 40 + 填充 20 + 素養閱讀 20 + 問答 20 = 100 分 |
| 高中 | 選擇 30 + 素養情境 40 + 問答 30 = 100 分 |

確認摘要：`科目 / 年級 / 範圍 / 題型組合 / 輸出：試卷 + 答案卷 .docx`

---

## Step 2：素養命題四要素

1. **真實情境**：題幹取材自生活、時事、本土文化（國小 50-150 字 / 國中 150-300 字 / 高中 250-500 字）
2. **整合運用**：每道素養題對應 ≥2 個學習表現代碼
3. **高階思維**：國中 ≥30% 達布魯姆 C4（分析）以上；高中 ≥50%
4. **多元呈現**：含圖表、非連續文本、對話情境、報告節錄等

|------|---------|---------|

## Step 4：題型格式規範

### 選擇題（四選一）
- 情境字數：國小 50-100 字 / 國中 100-200 字
- 標注難度（★☆☆/★★☆/★★★）+ 學習表現代碼
- 干擾選項合理，非明顯錯誤

### 素養閱讀理解（題組）
- 3-4 題共用一文本（250-500 字，改編自真實來源）
- 題目按難度遞增（字面理解→推論→評鑑應用）

### 非連續文本題
- 文字描述模擬：圖表/海報/菜單/統計圖等
- 測驗資訊擷取與判斷能力

### 問答/簡答
- 字數要求：國小 2-5 行 / 國中 50-100 字 / 高中 100-200 字
- 分項設計（1）具體問題（2）推理或評鑑

### 情境寫作（國語文）
- 提供真實寫作目的 + 假想讀者 + 情境說明
- 字數：國小 100-150 / 國中 200-350 / 高中 400-600
- 附評量規準簡表（4 向度 × 4 等第）

---

## Step 5：生成試卷文件

```bash
python scripts/generate_exam.py \
  --subject "[科目]" \
  --grade "[年級]" \
  --stage "[elementary/junior/senior]" \
  --range "[考試範圍]" \
  --types "[選擇:40,素養閱讀:20,問答:40]" \
  --output "/mnt/user-data/outputs/[科目]_[年級]_試卷.docx"
```

---

## 禁止事項（負規則）

| 禁止 | 原因 |
|------|------|
| 無情境純知識題（「什麼是比喻法？」）| 非素養導向，AI 完美作答 |
| 答案不唯一卻未標明 | 造成評分爭議 |
| 否定型題幹濫用 | 增加認知負擔，非必要不用 |
| 情境與題幹脫鉤 | 閱讀情境後還是能直接作答 |
| 難度全部相同 | 缺乏鑑別度，未符合布魯姆層次分布 |
| 學習表現代碼學段錯誤 | 國中用 -J- 非 -E- |

## Step 6：品質確認清單

- [ ] ≥50% 題目有真實生活情境（國中以上）
- [ ] 至少一道非連續文本或圖表解讀題
- [ ] 難度符合年級比例
- [ ] 每題標注難度（★）與布魯姆層次
- [ ] 每題標注學習表現代碼
- [ ] 試卷有封面、作答說明、各大題說明
- [ ] 答案卷含完整解析 + 補救教學建議
