## Step 1：年級偵測 + 引導式資訊收集

### 自動偵測年級
從使用者輸入嘗試辨識學習階段。若偵測成功，直接帶入後續；若失敗，執行以下問題。

### 第一輪（最多 3 個問題）
```
Q1: 考試範圍是什麼？（課文名稱、單元或主題）
Q2: 哪個年級？（如尚未偵測到）
Q3: 哪個科目？（如輸入不明確）
```

### 第二輪（選填）
```
Q4: 希望什麼題型組合？
    預設組合：
    ・國小：選擇 40 + 填充 20 + 問答 40 = 100 分
    ・國中：選擇 40 + 填充 20 + 素養閱讀 20 + 問答 20 = 100 分
    ・高中：選擇 30 + 素養情境 40 + 問答 30 = 100 分

Q5: 需要同時生成答案卷和解析嗎？（預設：是）
```

### 確認摘要（執行前輸出）
```
✅ 科目：{科目}
🎓 年級：{年級}（{學習階段}）
📚 範圍：{考試範圍}
📝 題型：{組合} ／ 共 100 分
📄 輸出：試卷 .docx + 答案卷 .docx
⚠️ 難度：{依年級自動套用分布}
```

## Step 2：素養情境命題四要素

1. **真實情境**：題幹取材自生活、時事、本土文化、科技社會議題
   - 情境字數：國小 50-150 字／國中 150-300 字／高中 250-500 字

2. **整合運用**：每道素養題對應至少 2 個學習表現代碼

3. **高階思維**：
   - 國中：至少 30% 題目達 C4（分析）以上
   - 高中：至少 50% 題目達 C4（分析）以上

4. **多元呈現**：含圖表、非連續文本、對話情境、報告節錄等

## Step 3：各科素養命題情境來源

### 國語文
情境類型：社會議題、臺灣文學、日常生活文書、跨域閱讀（科普/新聞）
題型：連續文本閱讀理解、非連續文本（廣告/海報）、語境選詞、情境寫作任務

### 數學
情境類型：消費決策（折扣/稅率）、環境數據（碳排/用水）、運動統計、城市規劃
題型：生活情境計算、圖表解讀、估算推理、多步驟整合

### 英語
情境類型：電子郵件、社群貼文、旅遊介紹、食譜說明、時刻表
題型：閱讀理解、情境對話、非連續文本（海報/活動說明）

### 社會
情境類型：史料分析、地圖判讀、時事新聞摘要、選舉數據
題型：史料分析、圖表解讀、議題判斷

### 自然
情境類型：氣候變遷、臺灣生態、科技發展（AI/生技）、健康生活
題型：科學探究設計、數據解讀、跨學科整合

## Step 4：各題型詳細規格

### 選擇題（四選一）
```
格式：
[情境段落]

X. [題幹]（難度★☆☆/★★☆/★★★｜對應學習表現：X-Ⅳ-X）
(A) [選項]  (B) [選項]  (C) [選項]  (D) [選項]

規則：
- 情境取材真實，避免無情境純知識測驗
- 干擾選項必須合理（非明顯錯誤）
- 避免否定型題幹（非必要不用）
```

### 素養閱讀理解（多題題組）
```
格式：
【題組 X】（共 X 分）

[閱讀文本 250-500 字，改編自真實來源，並標注：改編自《XX》XXXX 年]

X-1. [字面理解] ★☆☆    X-2. [推論判斷] ★★☆    X-3. [評鑑應用] ★★★

規則：
- 3-4 題為一題組，共用同一文本
- 題目按難度遞增
```

### 非連續文本題
```
格式：
[文字描述模擬圖表/海報/菜單/時刻表/統計圖等視覺材料]

X. 依據上方[材料名稱]，回答以下問題：...

規則：
- 文字描述需清楚模擬視覺材料內容
- 測驗資訊擷取與判斷能力
```

### 問答/簡答（含分項）
```
格式：
X. [情境描述]

請依據[情境/文本/材料]，回答以下問題：
（1）[較具體的分項問題]（X 分）
（2）[需推理或評鑑的分項問題]（X 分）

[教師版評分說明]

字數要求：國小 2-5 行／國中 50-100 字／高中 100-200 字
```

### 情境寫作（國語文）
```
格式：
六、作文（共 XX 分）

[提供真實寫作目的、假想讀者、情境說明]

例：「你的班級正在籌備一份給下一屆新生的『國中生活指南』，
請以『讓我們一起克服第一年的挑戰』為主題，寫 250-350 字的文章。」

附：評量規準簡表（4 向度 × 4 等第）
字數：國小 100-150 字／國中 200-350 字／高中 400-600 字
```

## Step 5：生成試卷文件

```bash
python scripts/generate_exam.py \
  --subject "[科目]" \
  --grade "[年級]" \
  --stage "[elementary/junior/senior]" \
  --range "[考試範圍]" \
  --types "[選擇:40,素養閱讀:20,問答:40]" \
  --output "/mnt/user-data/outputs/[科目]_[年級]_試卷.docx"
```

## Step 6：品質確認清單

- [ ] ≥ 50% 題目有真實生活情境（國中以上）
- [ ] 至少一道非連續文本或圖表解讀題
- [ ] 難度符合年級比例（見年級適應系統）
- [ ] 每題標注難度符號（★）與布魯姆層次
- [ ] 每題標注對應學習表現代碼
- [ ] 試卷有封面、作答說明、各大題說明
- [ ] 答案卷含完整解析 + 補救教學建議

## MCP 連接器（平台差異）

### Claude Code ／ Claude.ai（Pro/Team/Enterprise）
- **WebSearch**：搜尋最新時事素材作為情境題來源（自動啟用）
- **Google Drive**（選用）：直接儲存試卷到雲端硬碟

### Claude.ai（啟用方式）
Settings → Connectors → 啟用 Web Search + Google Drive

### Codex 平台
MCP Connectors 透過 `~/.codex/config.toml` 設定（`codex mcp add` 指令或手動編輯）。
未設定時自動降級：請參閱上方降級方案。

### Antigravity 平台（Google AI IDE）
MCP 透過 MCP Server Hub（1,500+ servers）或 `~/.gemini/antigravity/mcp_config.json` 設定。
支援 Jupyter Notebook 整合。未設定時自動降級：請參閱上方降級方案。
