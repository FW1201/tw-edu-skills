# K-12 教學套組客製化工具 v1.0

## 核心目的
「**讓每個 tw-edu-* Skill 都了解你的教學情境。**
不再每次都要重新說明你教幾年級、你在哪種學校。
設定一次，整套教學工具都變成你的專屬助理。」

---

## Step 0：讀取所有 tw-edu-* Skills 的 frontmatter

讀取以下 Skills 的可客製化參數（16 個核心教學 Skills）：
- `../tw-edu-lesson-plan-108/SKILL.md`
- `../tw-edu-exam-generator/SKILL.md`
- `../tw-edu-worksheet-creator/SKILL.md`
- （以及所有其他 tw-edu-* Skills）

## Step 2：系統性問卷（15 題）

### 教師基本資料
```
Q1：「你的稱呼？（讓 AI 稱呼你，例如：王老師）」
Q2：「你任教的學校類型？
    A. 公立國小     B. 私立國小
    C. 公立國中     D. 私立國中
    E. 公立高中     F. 私立高中
    G. 高職         H. 偏鄉/特殊學校」
Q3：「你主要教什麼科目？（可複選）
    國文 / 英語 / 數學 / 社會 / 自然
    藝術 / 體育 / 綜合 / 資訊 / 其他：___」
Q4：「你主要教哪些年段？
    低年段(1-2) / 中年段(3-4) / 高年段(5-6)
    國一 / 國二 / 國三 / 高一 / 高二 / 高三」
```

### 教學情境
```
Q5：「你的班級平均學生數？
    20 人以下 / 20-30 人 / 30 人以上」
Q6：「你的學生整體學習特性？
    A. 程度整齊，課業壓力大
    B. 程度差異大，需要差異化
    C. 偏鄉或資源不足
    D. 特殊教育需求學生較多
    E. 一般混合情況」
Q7：「你的學校有幾節彈性課程？
    （每週幾節可以自主規劃）」
Q8：「你使用哪種教科書版本？
    南一 / 翰林 / 康軒 / 自編 / 其他」
```

### 教學風格
```
Q9：「你的教學哲學偏向？
    A. 素養導向（核心素養連結優先）
    B. 知識建構（概念理解優先）
    C. 技能練習（反覆練習優先）
    D. 探究學習（PBL/問題導向）
    E. 混合型」
Q10：「你最常用的教學策略？（前三名）
     A. 直接教學    B. 合作學習    C. 討論教學
     D. 遊戲化      E. 翻轉教室    F. 差異化」
Q11：「你對素養題的態度？
     A. 非常重視，優先設計素養情境
     B. 平衡知識題與素養題
     C. 以知識題為主，素養為輔
     D. 還在摸索中」
```

### 行政與環境
```
Q12：「你擔任的角色？（可複選）
     □ 科任老師（只上課）
     □ 導師（有班級）
     □ 教學組長/主任
     □ 特教老師」
Q13：「你的學校是否有推行以下項目？
     □ PBL 專題式學習
     □ 校本課程
     □ 雙語教育
     □ 資訊融入教學
     □ 跨領域課程」
Q14：「你希望 AI 輸出的格式？
     A. 完整 Word 文件（學校要繳的格式）
     B. 結構化 Markdown（自己用）
     C. 簡要條列（快速參考）
     D. 依需求彈性」
Q15：「有什麼是你希望 AI 特別注意的？
     （例：我的學生都是原住民學生/我的班有很多過動學生）」
```

## Step 3：生成 teacher-profile.md

```markdown
# 教師設定檔
# 建立日期：[YYYY-MM-DD]
# tw-edu-synchronizer v1.0

---

## 教師基本資料
- 稱呼：[名字]
- 學校類型：[類型]
- 主要科目：[科目列表]
- 年段：[年段列表]

## 教學情境
- 班級人數：[人數]
- 學生特性：[特性描述]
- 彈性課程：每週 [N] 節
- 教科書版本：[版本]

## 教學風格偏好
- 教學哲學：[哲學取向]
- 常用策略：[策略列表]
- 素養題取向：[取向]

## 環境設定
- 教師角色：[角色列表]
- 學校推行項目：[項目列表]
- 輸出格式偏好：[格式]

## 各 Skill 客製化參數

### tw-edu-lesson-plan-108
- 預設年段：[年段]
- 預設科目：[科目]
- 教科書版本：[版本]
- 素養導向強度：[高/中/低]

### tw-edu-exam-generator
- 題型偏好：[選擇/問答/素養情境]
- 難度分布：[比例]
- 是否加入抗AI設計：[是/否]

### tw-edu-worksheet-creator
- 自主程度：[高鷹架/中/低鷹架]
- 使用時機：課前/課中/課後

### tw-edu-parent-communication
- 溝通語氣：[溫暖/正式/中性]
- 班級人數：[人數]
```

## Step 4：輸出已啟動的自訂項目

```
✅ 已客製化的項目（[N] 項）：

1. AI 稱呼你為「[稱呼]」
2. 所有教案預設年段：[年段]、科目：[科目]
3. 教科書版本：[版本]（自動對應課文）
4. 學生特性：[特性]（差異化設計自動啟用）
5. 輸出格式：[格式]
6. [特殊設定]已啟用

📁 設定檔已存入：./teacher-profile.md
   （所有 tw-edu-* Skills 啟動時會自動讀取此檔案）
```

---

## Step 5：更新設定

```
「更新某項設定」→ 只修改指定項目
「完全重設」→ 重新完成全部問卷
「查看目前設定」→ 顯示 teacher-profile.md 摘要
「分享設定給同事」→ 輸出可複製貼上的文字格式
```
