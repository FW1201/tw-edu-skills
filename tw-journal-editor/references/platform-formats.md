# 社群平台格式規範

數位敘事力期刊官方社群：FB 粉專 / Threads / Instagram
受眾：K-12 教師、教育研究者、教育行政人員、師培生

---

## 平台 A：Facebook 粉專

### 文案規範
| 項目 | 規範 |
|------|------|
| 字數 | 主文 400-800 字（附圖卡）；純文字可達 1,200 字 |
| 段落 | 每段 3-5 行，段間空行 |
| 開頭 | 第一行決定是否展開，必須有「鉤」 |
| 連結 | 連結放文末，前面先說為什麼值得點 |
| Hashtag | 3-5 個，放最後一段 |
| 圖卡 | 搭配輪播（3-5張）或單張封面圖 |

### FB 文案結構
```
【第一行鉤子】（讓人想按「查看更多」）
- 問句：「你有沒有試過...？」
- 數字：「3 個讓備課效率翻倍的 AI 工具」
- 情境：「昨天有個老師問我...」

【核心內容】（2-3 段）
段落一：問題或背景
段落二：方法或解方（可用清單）
段落三：具體成果或案例

【行動呼籲】（1段）
「你也可以試試...」或「有問題歡迎留言！」

【Hashtag 組】
#AI教育 #翻轉教育 #數位素養 #教師增能 #吳奇老師
```

### FB 最佳發布時間
- **週間**：週二、週四、週五 / 上午 8-10 時 or 下午 17-19 時
- **週末**：週六上午 9-11 時（教師週末滑手機高峰）
- **避開**：週一早上、週日晚上

## 平台 B：Threads

### 文案規範
| 項目 | 規範 |
|------|------|
| 單則字數 | 50-300 字（控制在 3 個捲動內） |
| 段落 | 單行換行，大量空行，視覺呼吸感 |
| 語氣 | 最口語、最直接，像在說話 |
| Hashtag | 0-2 個（Threads 不靠 hashtag 傳播） |
| 圖片 | 可搭配 1 張圖，不強制 |
| 連結 | Threads 連結不可點擊，改說「找我的 IG / 粉專」|

### Threads 文案風格
```
短句收尾，留白為王：

老師，
你有多久沒有好好備課了？

不是因為懶，
是因為根本沒有時間。

今天分享一個我用了三個月的方法，
讓我備課時間從兩小時變四十分鐘。

（以下是做法 ↓）

1. ...
2. ...
3. ...

試試看，下週告訴我效果。
```

### Threads 互動引爆點
- 用「留言+1 我分享給你」帶互動
- 結尾問一個具體問題（「你平常備課最花時間在哪？」）
- 敢說反直覺的立場（算法偏愛有觀點的貼文）

### Threads 最佳發布時間
- 平日：上午 7-8 時（通勤）、中午 12-13 時
- 傍晚：17-18 時（放學後）

## 平台 C：Instagram

### 文案規範
| 項目 | 規範 |
|------|------|
| 圖卡尺寸 | 1080×1080px（正方形）首選；1080×1350px 亦可 |
| 輪播張數 | 3-10 張，首張決定是否點開 |
| 說明文字 | 150-300 字 + hashtag 組 |
| Hashtag | 10-20 個（IG 靠 hashtag 觸及） |
| IG Story | 搭配 Sticker 投票/問答（引導互動）|

### IG Hashtag 策略（分三層）
```
【大標籤，觸及廣】（5-8個）
#教育 #台灣教育 #老師 #AI教育 #數位教育

【中標籤，精準觸及】（5-8個）
#翻轉教育 #數位素養 #教師增能 #AI工具 #ChatGPT教育

【小標籤，社群認同】（3-5個）
#數位敘事力 #吳奇老師 #AITeacher #VibeCode教育
```

### 輪播圖卡順序邏輯
1. **封面**：強標題 + 視覺衝擊（讓人划停）
2. **問題卡**：「你是不是也有這個困擾？」
3. **解法卡 1-2**：核心做法
4. **案例/成果卡**：「這樣做之後...」
5. **行動卡**：「下載 / 試試看 / 追蹤我們」

## 本月排程模板（每月 4 週）

```
第一週（新文章發布週）
  週一：Threads 預告（短文引發期待）
  週三：FB + IG 正式發布（主文 + 輪播圖卡）
  週五：FB 互動貼文（問讀者感想 / 延伸問題）

第二週（深化討論週）
  週二：Threads 補充（文章沒說到的 1 個觀點）
  週四：IG Story 問答（讀者問題 Q&A）

第三週（資源分享週）
  週一：推薦工具 / 延伸閱讀（較輕量）
  週四：「一張圖看懂...」快速知識卡

第四週（回顧準備週）
  週二：本月精華整理（Threads）
  週五：下月預告（吊胃口）
```

## 成效追蹤指標

### 核心指標（每篇必追）
| 指標 | 說明 | 目標基準 |
|------|------|---------|
| 觸及人數 | 貼文被看到幾次 | 環比成長 5% |
| 互動率 | （按讚+留言+分享）/ 觸及 | FB ≥ 2%；IG ≥ 3% |
| 儲存數（IG）| 被儲存代表高價值內容 | 每篇 ≥ 30 次 |
| 連結點擊 | 驅動到文章閱讀 | 每次 ≥ 50 次 |

### 每月成效回顧問題
1. **本月最高觸及**：哪篇？標題是什麼？（標題值得複製）
2. **本月最高互動**：哪篇？什麼格式？（格式值得重複）
3. **本月最多儲存**：哪張圖卡？（設計邏輯值得保留）
4. **踩坑紀錄**：哪篇表現最差？為什麼？

### 成效記錄格式（寫入 Claude Memory）
```
數位敘事力社群成效（YYYY/MM）：
最高觸及：[主題] [平台] [觸及數]
最佳格式：[輪播/單圖/純文字]
下月優化方向：[1-2個調整]
```
