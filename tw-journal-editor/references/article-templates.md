# 文章格式模板

## 格式 A：翻轉教育投稿格式

### 基本規格
| 項目 | 規範 |
|------|------|
| 字數 | 正文 1,200-2,000 字（含小標題，不含標題和作者） |
| 標題 | 25 字以內，含關鍵詞 |
| 語氣 | 第一人稱為主，親切實用，避免學術腔 |
| 圖片 | 1-3 張，附圖說，寬度 ≥ 1200px |
| 作者介紹 | 50字以內，含職稱、學校/機構、代表成就 |
| 段落結構 | 每段 150-250 字，大量使用小標題分段 |

### 文章結構（標準版）

```
【標題】（A/B 兩版備用）

【作者介紹】
吳奇老師，現任○○，數位敘事力期刊主編，
Google 認證教育家，著有...

【引言鉤子】（150-200字）
用一個讓讀者「咦，這不就是我嗎！」的開場：
- 描述教師熟悉的困境或情境
- 提出一個反直覺的觀點
- 用驚人數據開場
結尾一句點出本文解決什麼問題。

【小標題一：問題診斷】（300-400字）
你一直以為是這樣，但其實...
- 具體描述問題的根源
- 可用 2-3 個具體例子

【小標題二：核心做法】（400-500字）
具體可操作的方法，用步驟或清單呈現
- 步驟 1：...
- 步驟 2：...
- 步驟 3：...
配合工具或案例說明

【小標題三：實際案例】（250-350字）
「我自己這樣做之後...」或「○○老師分享...」
- 具體成果（數字、學生反應、課堂變化）
- 踩過的坑和調整過程

【小標題四：給老師的行動建議】（200-300字）
讀者讀完後「明天就能做的事」：
1. 立刻：___（今天就能完成）
2. 本週：___（這週試試看）
3. 本學期：___（長期規劃）

【結語】（100-150字）
呼應引言，說出核心信念，加上呼籲行動
（例：「現在就打開你的電腦，試試看...」）
```

### 標題公式庫

| 公式 | 範例 |
|------|------|
| 數字型 | 「3 個讓 AI 幫你改作業的方法，老師真的省下 40%時間」 |
| 問句型 | 「學生都在用 AI 寫報告？老師更應該先學！」 |
| 反差型 | 「不懂程式的老師，怎麼做出 4836 次下載的 Chrome Extension？」 |
| 解方型 | 「AI 備課不用從零開始：用這個流程，30分鐘備完一節課」 |
| 身份認同型 | 「身為台灣老師，我為什麼選擇擁抱 AI 而不是抗拒它」 |

### 禁忌清單（翻轉教育）

- 不用「人工智慧」，用「AI」
- 不寫長於 3 行的段落（沒有小標題）
- 不寫沒有具體數字的空話（「效率大幅提升」→ 改成「備課時間從 2 小時縮短到 40 分鐘」）
- 不引用找不到原始來源的數據
- 不用「筆者認為」或「本文旨在」這類學術腔

## 格式 B：數位敘事力期刊格式

### 基本規格
| 項目 | 規範 |
|------|------|
| 字數 | 正文 2,000-4,000 字 |
| 標題 | 30 字以內 |
| 語氣 | 第一人稱，帶學術骨幹但口語表達 |
| 摘要 | 200-300 字，含關鍵詞 5-8 個 |
| 參考文獻 | APA 7th 格式，至少 5 篇 |
| 圖表 | 研究架構圖或數據圖表（建議） |

### 文章結構（半學術版）

```
【標題】

【摘要】（200-300字）
背景 → 目的 → 方法/做法 → 主要發現 → 意涵

【關鍵詞】（5-8個，中文）

【一、前言】（300-500字）
問題背景 → 研究/實踐缺口 → 本文目的 → 文章結構說明

【二、理論框架 / 背景文獻】（500-800字）
相關理論或工具背景介紹
至少引用 3 篇文獻（調用 Consensus MCP 查找）

【三、實踐設計 / 方法】（600-900字）
具體做了什麼、怎麼設計、用了什麼工具
可含流程圖（調用 Excalidraw MCP）

【四、成果與反思】（500-700字）
具體結果描述（量化 + 質化）
遇到的挑戰與調整

【五、結論與建議】（300-400字）
回應前言的缺口
給讀者的建議
後續研究/實踐方向

【參考文獻】（APA 7th）
```

## 快速引言鉤子模板

複製使用，填入具體內容：

**情境型**
> 「上個月的期中考批改完，我面對著 35 份作文，心想：這 35 個學生，我真的能給出 35 個不同的建議嗎？」

**數據型**
> 「根據一項對台灣 500 位教師的調查，73% 的人每週花在行政與備課上的時間超過 20 小時。這讓我開始思考：AI 能幫忙解決哪一塊？」

**反直覺型**
> 「我曾經認為，讓 AI 幫忙出題是在偷懶。直到我親自用 Claude 生成了一套素養題，才發現：真正的挑戰不是生成，而是審題。」

**對話型**
> 「有個學生問我：『老師，你有沒有試過用 AI 幫你改作業？』我當時回答沒有。但這個問題讓我睡不著覺。」
